Étapes pour installer Django

  1. Installer Python
  2. Installer Django avec la commande pip install django
  3. Créer un projet Django avec django-admin startproject myproject
  4. Lancer le serveur avec python manage.py runserver
  5. Accéder à l'application sur http://127.0.0.1:8000/
